# React Task Board with Authentication

A simple task management board built with **React and TypeScript**.  
The application allows users to create, manage, and organize tasks across multiple stages of completion.

This project demonstrates core **React development concepts**, including component-based architecture, state management, higher-order components, and responsive design.

---

## Features

- Login-protected task creation using a **Higher Order Component (HOC)**
- Add new tasks with title and priority
- Move tasks between stages:
  - To Do
  - In Progress
  - Done
- Delete tasks
- Sort tasks by priority
- Responsive layout for desktop and mobile
- Clean UI with a dark theme

---

## Technologies Used

- **React**
- **TypeScript**
- **Vite**
- **CSS (Custom Styling)**
- Component-based architecture
